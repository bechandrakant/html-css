steps to meet standards

  1. DOCTYPE Definition
  2. Multipurpose Internet Mail Extensions (MIME)
  3. All tags must nest properly and not overlap
  4. Use all lowercase for element names
  5. Always use closing tags
  6. Empty elements are marked with closing slash
  7. Attribute values must be contained in quotation marks
